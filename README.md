原理：使用Main.py来启动EasiNote5

使用教程：


         1.安装python 
         Add PATH

         2.安装依赖 
         运行dependency.bat即可
         
         3.修改CustomSplasherConfig.json中target_program为软件目标路径 
         默认为EasiNote5默认安装位置

         4.默认使用python打开Main.py即可
